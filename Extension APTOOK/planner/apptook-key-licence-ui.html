APPTOOK Backend Concept

สร้าง APPTOOK key, import licence ทีละรายการ และผูกหลาย licence ต่อ key ได้

Mockup นี้แสดง flow ของระบบหลังบ้านตามหลักการใหม่: 1) กำหนด APPTOOK key 2) import licence ทีละรายการเพื่อใส่รายละเอียดได้ครบ 3) เลือก licence ได้มากกว่า 1 รายการจากรายการที่ import ไว้แล้ว 4) assign / unassign / replace mapping ได้อย่างชัดเจน

Step 1

Prepare the licence pool

Add each licence to the central pool first so it is ready to be assigned to a key in the next step.

Quick add
เมื่อกด add จะถูกตั้งสถานะเป็น available ทันที

Step 2

Create an APPTOOK key

Create a new key for the user, then set the status and expiry before linking any licences.

Primary key

Step 3

Assign one key to multiple licences

Select a key first, then choose multiple licences from the same pool and assign them from one screen.

Multi-assign workflow
ติ๊กเลือกได้หลายรายการโดยไม่ต้องกด Ctrl / Cmd

แนะนำ flow แบบง่าย: 1) เลือก key 2) กรอง licence 3) เลือกหลายรายการ 4) กด assign 5) ถ้าต้องการแก้ไข ใช้ unassign หรือ replace
